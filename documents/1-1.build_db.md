# 1-1. PostgreSQL 로컬 DB 구축

## 설치 정보

| 항목 | 값 |
|------|-----|
| DBMS | PostgreSQL 17.9 |
| OS | Ubuntu 24.04.4 LTS |
| 설치 방식 | APT (공식 PostgreSQL 저장소) |
| 포트 | 5432 (기본값) |
| 인코딩 | UTF-8 |
| 로케일 | ko_KR.UTF-8 |
| 설정 파일 | `/etc/postgresql/17/main/postgresql.conf` |
| HBA 파일 | `/etc/postgresql/17/main/pg_hba.conf` |
| 데이터 디렉토리 | `/var/lib/postgresql/17/main` |

## 계정 정보

| 항목 | 값 |
|------|-----|
| 사용자명 | `glovis` |
| 비밀번호 | `.env` 파일 참조 (`DATABASE_PASSWORD`) |
| 권한 | SUPERUSER, CREATEDB, CREATEROLE |
| 소유 DB | `gapttuk` |

> ⚠️ 비밀번호는 이 문서에 기록하지 않습니다. `.env` 파일에서 관리합니다.

## 접속 방법

```bash
# CLI 접속 (.env에서 비밀번호 참조)
# .pgpass 파일 권장 (프로세스 리스트 노출 방지)
# ~/.pgpass 형식: localhost:5432:gapttuk:glovis:<password>
psql -U glovis -d gapttuk -h localhost

# 또는 Connection String (.env 파일)
# DATABASE_URL=postgresql://glovis:<password>@localhost:5432/gapttuk
```

## 서비스 관리

```bash
# 시작
sudo systemctl start postgresql

# 중지
sudo systemctl stop postgresql

# 재시작
sudo systemctl restart postgresql

# 상태 확인
sudo pg_isready
```
